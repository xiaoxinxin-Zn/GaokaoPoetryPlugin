# 更新日志

## v1.2.0.0 (2026-05-31)

### 数据
- 补全 60 篇全覆盖，每篇不少于 4 句（共 251 条名句）
- 所有 content 已与 poetry.json 原文精确对齐，修复标点/错字 35 处
- 新增易错字数据库 `Assets/typo_prone_chars.json`，收录 22 组高考高频易错字

### 改进
- 修复数据加载竞态条件：先等待数据就绪再启动轮播
- 异常不再静默吞没，UI 中显示具体错误信息
- `PoetryService` 注册到 DI 容器
- 数据校验脚本 `validate_data.py`：支持构建时自动校验（`--ci` 模式）
- `download_data.py` 支持 `--validate` 自动校验下载的数据

### 构建
- 构建时自动运行数据校验，不一致时发出 MSBuild Warning

## v1.1.0.0 (2026-05-04)

### 新增
- 初始版本发布
- 随机展示高考必背60篇古诗文名句
- 支持按教材册别筛选：必修上、必修下、选择性必修上、选择性必修中、选择性必修下、其他
- 支持显示/隐藏作者与篇名
- 可调节轮播间隔与字体大小
- 嵌入数据离线可用，网络加载作为备选
- 项目主页迁移至 GitHub

### 数据来源
- 古诗文数据来自 [gaokao-poetry](https://clover-yan.github.io/gaokao-poetry/)（CC BY-SA 4.0）
